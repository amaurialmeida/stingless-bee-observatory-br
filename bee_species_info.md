# Informações sobre Abelhas Sem Ferrão (Meliponini) no Brasil

## Raios de Forrageamento (Voo) Estimados

| Espécie | Nome Científico | Raio de Voo (m) | Observações |
| :--- | :--- | :--- | :--- |
| **Jataí** | *Tetragonisca angustula* | 500 - 600 | Espécie pequena, muito comum em áreas urbanas. |
| **Mandaçaia** | *Melipona quadrifasciata* | 2000 | Espécie maior, robusta. Algumas fontes citam até 2km. |
| **Uruçu** | *Melipona scutellaris* | 1500 - 2000 | Comum no Nordeste, grande porte. |
| **Manduri** | *Melipona asilvai* | 1500 | Porte médio/grande. |
| **Mirim** | *Plebeia spp.* | 500 | Abelhas muito pequenas. |
| **Tubuna** | *Scaptotrigona bipunctata* | 1000 | Agressiva, porte médio. |
| **Tiúba** | *Melipona fasciculata* | 2000 | Comum no Maranhão e Pará. |
| **Irapuá** | *Trigona spinipes* | 1000 - 1500 | Muito comum, generalista. |

## Fontes de Dados para Ocorrências
- **GBIF**: Global Biodiversity Information Facility (API disponível).
- **iNaturalist**: Observações de cidadãos cientistas.
- **SpeciesLink**: Rede de dados biológicos do Brasil.

## Estética do Projeto
- **Mapa**: Estilo bolhas (bubble map) inspirado no painel COVID-19 do Ministério da Saúde.
- **Círculos**: Sobreposição de círculos de forrageamento (buffer) no mapa.
- **Tecnologias**: Streamlit, Folium, Plotly, Pandas, Geopandas.
